A fairly simple shop display which pulls item lists and items from a JSON file and auto-populates the left section with each list and the items in them. Click an item, and the right section will display the associated properties of the item. I want to include a 'cart' and buttons to add certain quanitities of each item, but that's on the backburner while I make other components.
